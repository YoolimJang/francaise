프랑스어 동사 중 가장 많고 규칙적인 무리다. 어간에 아래 어미를 붙여 현재형(présent)을 만든다.

## 현재형 어미
어간 + 어미: **-e, -es, -e, -ons, -ez, -ent**

| 인칭 | 어미 | 예: parler |
|---|---|---|
| je | -e | parle |
| tu | -es | parles |
| il/elle | -e | parle |
| nous | -ons | parlons |
| vous | -ez | parlez |
| ils/elles | -ent | parlent |

- 어미 -e, -es, -ent는 **묵음**이라 je / tu / il / ils 발음이 같다.
- 일부는 철자 변화: préférer(é→è), acheter(e→è), manger·voyager(nous mangeons / voyageons), payer·essayer(y→i).

## 대표 1군 동사 21개

| 원형 | 뜻 | 예문 |
|---|---|---|
| parler | 말하다 | Je parle français. (나는 프랑스어를 말한다.) |
| habiter | 살다 | J'habite à Paris. (나는 파리에 산다.) |
| adorer | 아주 좋아하다 | J'adore le chocolat. (나는 초콜릿을 아주 좋아한다.) |
| travailler | 일하다 | Je travaille à la maison. (나는 집에서 일한다.) |
| rester | 머무르다, 남다 | Je reste à la maison ce soir. (나는 오늘 저녁 집에 머무른다.) |
| préférer | 선호하다 | Je préfère le thé au café. (나는 커피보다 차를 더 좋아한다.) |
| détester | 싫어하다 | Je déteste le lundi. (나는 월요일을 싫어한다.) |
| jouer | 놀다 / (게임·악기를) 하다 | Je joue au foot. (나는 축구를 한다.) |
| regarder | 보다, 바라보다 | Je regarde la télé. (나는 텔레비전을 본다.) |
| écouter | 듣다 | J'écoute de la musique. (나는 음악을 듣는다.) |
| cuisiner | 요리하다 | Je cuisine ce soir. (나는 오늘 저녁에 요리한다.) |
| voyager | 여행하다 | Je voyage en train. (나는 기차로 여행한다.) |
| chanter | 노래하다 | Je chante sous la douche. (나는 샤워하면서 노래한다.) |
| manger | 먹다 | Je mange une pomme. (나는 사과를 먹는다.) |
| aider | 돕다 | J'aide ma mère. (나는 엄마를 돕는다.) |
| acheter | 사다 | J'achète du pain à la boulangerie. (나는 빵집에서 빵을 산다.) |
| payer | 지불하다 | Je paie par carte. (나는 카드로 지불한다.) |
| essayer | 시도하다, 입어보다 | J'essaie cette veste. (나는 이 재킷을 입어본다.) |
| chercher | 찾다 | Je cherche mes clés. (나는 내 열쇠를 찾는다.) |
| traverser | 건너다, 가로지르다 | Je traverse la rue. (나는 길을 건넌다.) |
| tourner | 돌다, 돌리다, 방향을 틀다 | Tourne à droite. (오른쪽으로 돌아.) |

## 동사별 활용형 (구문)
1군 동사 중에는 뒤에 **동사원형**이나 특정 **전치사**를 붙여 쓰는 것들이 있다.

- **aimer / adorer / préférer / détester + 동사원형** — ~하기를 좋아하다 / 아주 좋아하다 / 선호하다 / 싫어하다
  - J'**aime voyager**. (나는 여행하기를 좋아한다.) · Je **déteste attendre**. (나는 기다리기 싫다.)
- **essayer de + 동사원형** — ~하려고 애쓰다
  - J'**essaie de comprendre**. (나는 이해하려고 애쓴다.)
- **jouer à + 게임·스포츠 / jouer de + 악기** — ~을 하다 / (악기를) 연주하다
  - Je **joue au** foot. (축구를 한다.) · Je **joue du** piano. (피아노를 친다.)
