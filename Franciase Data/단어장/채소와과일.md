# 채소와 과일 (Les légumes et les fruits)

## 채소 (les légumes)
**la tomate** (f) /tɔ.mat/ 토마트 — 토마토
- J'achète des tomates. — 토마토를 사.
- La tomate est rouge. — 토마토는 빨개.

**la carotte** (f) /ka.ʁɔt/ 카로트 — 당근
- Le lapin mange une carotte. — 토끼가 당근을 먹어.
- Je coupe les carottes. — 당근을 썰어.

**la pomme de terre** (f) /pɔm də tɛʁ/ 폼 드 테르 — 감자
- Je fais des pommes de terre. — 감자 요리를 해.
- J'aime les pommes de terre. — 나는 감자를 좋아해.

**l'oignon** (m) /ɔ.ɲɔ̃/ 오뇽 — 양파
- Je coupe un oignon. — 양파를 썰어.
- L'oignon fait pleurer. — 양파는 눈물 나게 해.

**la salade** (f) /sa.lad/ 살라드 — 상추, 샐러드
- Je mange une salade. — 샐러드를 먹어.
- La salade est fraîche. — 샐러드가 신선해.

**le champignon** (m) /ʃɑ̃.pi.ɲɔ̃/ 샹피뇽 — 버섯
- J'aime les champignons. — 나는 버섯을 좋아해.
- Des champignons dans la sauce. — 소스에 버섯이 들어가.

**le concombre** (m) /kɔ̃.kɔ̃bʁ/ 콩콩브르 — 오이
- Une salade de concombre. — 오이 샐러드.
- Le concombre est vert. — 오이는 초록색이야.

**le poivron** (m) /pwa.vʁɔ̃/ 푸아브롱 — 피망
- Un poivron rouge. — 빨간 피망.
- Je cuisine des poivrons. — 피망을 요리해.

**l'ail** (m) /aj/ 아유 — 마늘
- Je mets de l'ail. — 마늘을 넣어.
- L'ail sent fort. — 마늘은 냄새가 강해.

## 과일 (les fruits)
**la pomme** (f) /pɔm/ 폼 — 사과
- Je mange une pomme. — 사과를 먹어.
- Les pommes sont bonnes. — 사과가 맛있어.

**la banane** (f) /ba.nan/ 바난 — 바나나
- J'adore les bananes. — 나는 바나나를 정말 좋아해.
- La banane est jaune. — 바나나는 노래.

**l'orange** (f) /ɔ.ʁɑ̃ʒ/ 오랑주 — 오렌지
- Je bois du jus d'orange. — 오렌지 주스를 마셔.
- L'orange est un fruit. — 오렌지는 과일이야.

**la fraise** (f) /fʁɛz/ 프레즈 — 딸기
- J'aime les fraises. — 나는 딸기를 좋아해.
- Des fraises avec de la crème. — 크림 얹은 딸기.

**le raisin** (m) /ʁɛ.zɛ̃/ 레쟁 — 포도
- Le raisin est sucré. — 포도가 달아.
- J'achète du raisin. — 포도를 사.

**la pêche** (f) /pɛʃ/ 페슈 — 복숭아
- Une pêche juteuse. — 즙 많은 복숭아.
- J'aime les pêches en été. — 여름에 복숭아를 좋아해.

**le citron** (m) /si.tʁɔ̃/ 시트롱 — 레몬
- Un citron jaune. — 노란 레몬.
- Je mets du citron dans le thé. — 차에 레몬을 넣어.

**la poire** (f) /pwaʁ/ 푸아르 — 배
- Je mange une poire. — 배를 먹어.
- La poire est mûre. — 배가 익었어.

**le melon** (m) /mə.lɔ̃/ 믈롱 — 멜론
- Le melon est sucré. — 멜론이 달아.
- Une tranche de melon. — 멜론 한 조각.
