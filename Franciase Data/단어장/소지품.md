# 소지품·사물 (Les affaires)

**le sac** (m) /sak/ 사크 — 가방
- J'ai un sac noir. — 검은 가방이 있어.
- Ton sac est joli. — 네 가방 예쁘다.

**le sac à dos** (m) /sak a do/ 사카도 — 배낭
- Je porte un sac à dos. — 배낭을 메고 있어.
- Mon sac à dos est lourd. — 내 배낭은 무거워.

**le portefeuille** (m) /pɔʁ.tə.fœj/ 포르트푀유 — 지갑
- Où est mon portefeuille ? — 내 지갑 어디 있지?
- Mon portefeuille est vide. — 지갑이 비었어.

**les clés** (f.pl) /kle/ 클레 — 열쇠
- J'ai perdu mes clés. — 열쇠를 잃어버렸어.
- Voici les clés. — 여기 열쇠 있어.

**le téléphone** (m) /te.le.fɔn/ 텔레폰 — 전화기 ≈ telephone
- Mon téléphone sonne. — 전화가 울려.
- Je cherche mon téléphone. — 전화기를 찾고 있어.

**le portable** (m) /pɔʁ.tabl/ 포르타블 — 휴대폰 ≈ portable
- Mon portable est déchargé. — 휴대폰 배터리가 나갔어.
- Je regarde mon portable. — 휴대폰을 봐.

**l'ordinateur** (m) /ɔʁ.di.na.tœʁ/ 오르디나퇴르 — 컴퓨터
- Je travaille sur mon ordinateur. — 컴퓨터로 일해.
- Mon ordinateur est lent. — 내 컴퓨터는 느려.

**les lunettes** (f.pl) /ly.nɛt/ 뤼네트 — 안경
- Je porte des lunettes. — 나는 안경을 써.
- Où sont mes lunettes ? — 내 안경 어디 있지?

**le stylo** (m) /sti.lo/ 스틸로 — 볼펜
- J'écris avec un stylo. — 볼펜으로 써.
- Tu as un stylo ? — 볼펜 있어?

**le crayon** (m) /kʁɛ.jɔ̃/ 크레용 — 연필 ≈ crayon
- Je dessine avec un crayon. — 연필로 그려.
- Prête-moi ton crayon. — 연필 좀 빌려줘.

**le cahier** (m) /ka.je/ 카예 — 공책
- J'ouvre mon cahier. — 공책을 펴.
- Mon cahier est plein. — 공책이 꽉 찼어.

**le livre** (m) /livʁ/ 리브르 — 책
- Je lis un livre. — 책을 읽어.
- C'est un bon livre. — 좋은 책이야.

**le parapluie** (m) /pa.ʁa.plɥi/ 파라플뤼 — 우산
- Prends ton parapluie, il pleut. — 비 와, 우산 챙겨.
- J'ai oublié mon parapluie. — 우산을 깜빡했어.

**la montre** (f) /mɔ̃tʁ/ 몽트르 — 손목시계
- J'ai une nouvelle montre. — 새 시계가 있어.
- Ma montre est cassée. — 내 시계가 고장 났어.

**le journal** (m) /ʒuʁ.nal/ 주르날 — 신문 ≈ journal
- Je lis le journal le matin. — 아침에 신문을 읽어.
- Le journal est sur la table. — 신문이 탁자 위에 있어.

**le cadeau** (m) /ka.do/ 카도 — 선물
- C'est un cadeau pour toi. — 너를 위한 선물이야.
- J'adore ce cadeau. — 이 선물 정말 좋아.

**la chose** (f) /ʃoz/ 쇼즈 — 것, 물건
- J'ai une chose à te dire. — 너한테 할 말이 있어.
- C'est une belle chose. — 좋은 일이야.

**le verre** (m) /vɛʁ/ 베르 — 잔, 유리
- Un verre d'eau, s'il vous plaît. — 물 한 잔 주세요.
- Le verre est plein. — 잔이 가득 찼어.

**le canapé** (m) /ka.na.pe/ 카나페 — 소파
- Je m'assois sur le canapé. — 소파에 앉아.
- Le chat dort sur le canapé. — 고양이가 소파에서 자.

**le soleil** (m) /sɔ.lɛj/ 솔레유 — 해, 햇빛
- Il y a du soleil aujourd'hui. — 오늘 해가 나.
- J'aime le soleil. — 나는 햇빛을 좋아해.
