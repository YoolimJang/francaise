# 사물·사람 묘사 형용사 (Les adjectifs de description)

성·수 일치 규칙은 [[12-형용사]]. 감정 형용사는 [[감정과상태]] 참고.

**grand / grande** /ɡʁɑ̃/ 그랑 — 큰, 키 큰
- C'est un grand homme. — 키 큰 남자야.
- Elle habite dans une grande maison. — 그녀는 큰 집에 살아.

**petit / petite** /pə.ti/ 프티 — 작은
- J'ai un petit chat. — 작은 고양이가 있어.
- C'est une petite fille. — 어린 소녀야.

**gros / grosse** /ɡʁo/ 그로 — 뚱뚱한, 굵은
- Un gros chien. — 덩치 큰 개.
- J'ai une grosse valise. — 큰 여행 가방이 있어.

**long / longue** /lɔ̃/ 롱 — 긴
- C'est un long voyage. — 긴 여행이야.
- Quelle longue journée ! — 긴 하루였어!

**court / courte** /kuʁ/ 쿠르 — 짧은
- Il a les cheveux courts. — 그는 머리가 짧아.
- Elle porte une jupe courte. — 그녀는 짧은 치마를 입어.

**beau / belle** /bo/ 보 — 아름다운
- C'est un beau jardin. — 아름다운 정원이야.
- Quelle belle vue ! — 경치 멋지다!

**joli / jolie** /ʒɔ.li/ 졸리 — 예쁜
- Un joli sac. — 예쁜 가방.
- Elle porte une jolie robe. — 그녀는 예쁜 원피스를 입어.

**jeune** /ʒœn/ 죈 — 젊은
- C'est un homme jeune. — 젊은 남자야.
- Elle est encore jeune. — 그녀는 아직 젊어.

**vieux / vieille** /vjø/ 비외 — 늙은, 오래된
- J'ai un vieux livre. — 오래된 책이 있어.
- Une vieille voiture. — 낡은 차.

**nouveau / nouvelle** /nu.vo/ 누보 — 새로운
- J'ai un nouveau téléphone. — 새 전화기가 있어.
- Ils ont une nouvelle maison. — 그들은 새 집이 있어.

**bon / bonne** /bɔ̃/ 봉 — 좋은, 맛있는
- C'est un bon restaurant. — 좋은 식당이야.
- Quelle bonne idée ! — 좋은 생각이야!

**mauvais / mauvaise** /mo.vɛ/ 모베 — 나쁜
- C'est un mauvais film. — 나쁜 영화야.
- Il fait mauvais aujourd'hui. — 오늘 날씨가 나빠.

**cher / chère** /ʃɛʁ/ 셰르 — 비싼, 소중한
- Ce sac est cher. — 이 가방은 비싸.
- C'est trop cher. — 너무 비싸.

**facile** /fa.sil/ 파실 — 쉬운
- C'est un exercice facile. — 쉬운 문제야.
- Le français est facile ! — 프랑스어는 쉬워!

**difficile** /di.fi.sil/ 디피실 — 어려운
- C'est un livre difficile. — 어려운 책이야.
- La grammaire est difficile. — 문법이 어려워.

**gentil / gentille** /ʒɑ̃.ti/ 장티 — 친절한
- C'est un homme gentil. — 친절한 남자야.
- Elle est très gentille. — 그녀는 아주 친절해.

**intéressant / intéressante** /ɛ̃.te.ʁɛ.sɑ̃/ 앵테레상 — 흥미로운
- C'est un livre intéressant. — 흥미로운 책이야.
- Ton idée est intéressante. — 네 생각 흥미롭다.

**lourd / lourde** /luʁ/ 루르 — 무거운
- Ce sac est lourd. — 이 가방은 무거워.
- Une valise lourde. — 무거운 여행 가방.

**léger / légère** /le.ʒe/ 레제 — 가벼운
- Un manteau léger. — 가벼운 코트.
- Cette veste est légère. — 이 재킷은 가벼워.

**génial / géniale** /ʒe.njal/ 제니알 — 멋진, 훌륭한
- C'est génial ! — 멋지다!
- Quelle idée géniale ! — 정말 좋은 생각이야!

**magnifique** /ma.ɲi.fik/ 마니피크 — 아주 아름다운
- Quelle vue magnifique ! — 경치가 정말 아름다워!
- C'est magnifique. — 멋지다.

**incroyable** /ɛ̃.kʁwa.jabl/ 앵크루아야블 — 믿기 힘든
- C'est une histoire incroyable. — 믿기 힘든 이야기야.
- C'est incroyable ! — 믿을 수 없어!

**amusant / amusante** /a.my.zɑ̃/ 아뮈장 — 재미있는
- C'est un jeu amusant. — 재미있는 놀이야.
- La soirée était amusante. — 파티가 재밌었어.

**vrai / vraie** /vʁɛ/ 브레 — 진짜의
- C'est une vraie histoire. — 진짜 이야기야.
- C'est vrai ? — 진짜야?

**cuit / cuite** /kɥi/ 퀴 — 익힌
- Je mange de la viande cuite. — 익힌 고기를 먹어.
- Les légumes sont cuits. — 채소가 익었어.

**grave** /ɡʁav/ 그라브 — 심각한
- C'est un problème grave. — 심각한 문제야.
- Ce n'est pas grave. — 괜찮아.

**mince** /mɛ̃s/ 망스 — 얇은, 날씬한
- Une tranche mince. — 얇은 조각.
- Elle est mince. — 그녀는 날씬해.

**blond / blonde** /blɔ̃/ 블롱 — 금발의
- C'est un garçon blond. — 금발 소년이야.
- Elle est blonde. — 그녀는 금발이야.

> [!note] 명사 앞에 오는 형용사
> beau, joli, grand, petit, bon, vieux, nouveau는 예외적으로 명사 **앞**: un **petit** chat.
