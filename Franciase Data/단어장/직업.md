# 직업 (La profession)

être 뒤에서 관사 없이 씀: Je suis étudiant. 성에 따라 여성형 변화([[11-명사]]).

**étudiant / étudiante** /e.ty.djɑ̃/ 에튀디앙 — 학생
- Je suis étudiant. — 나는 학생이야.
- Elle est étudiante en droit. — 그녀는 법학과 학생이야.

**professeur / professeure** /pʁɔ.fɛ.sœʁ/ 프로페쇠르 — 교사, 교수 ≈ professor
- Mon père est professeur. — 우리 아빠는 교사야.
- La professeure est gentille. — 선생님은 다정해.

**médecin** (m) /med.sɛ̃/ 메드쌍 — 의사 ≈ medic
- Elle est médecin. — 그녀는 의사야.
- Je vais chez le médecin. — 나는 병원에 가.

**infirmier / infirmière** /ɛ̃.fiʁ.mje/ 앵피르미에 — 간호사 ≈ infirmary
- Il est infirmier. — 그는 간호사야.
- L'infirmière est très gentille. — 간호사는 아주 친절해.

**avocat / avocate** /a.vɔ.ka/ 아보카 — 변호사
- Ma sœur est avocate. — 내 누나는 변호사야.
- Il travaille comme avocat. — 그는 변호사로 일해.

**ingénieur / ingénieure** /ɛ̃.ʒe.njœʁ/ 앵제니외르 — 엔지니어 ≈ engineer
- Je suis ingénieur. — 나는 엔지니어야.
- Elle est ingénieure. — 그녀는 엔지니어야.

**cuisinier / cuisinière** /kɥi.zi.nje/ 퀴지니에 — 요리사
- Mon oncle est cuisinier. — 삼촌은 요리사야.
- Elle est bonne cuisinière. — 그녀는 요리를 잘해.

**serveur / serveuse** /sɛʁ.vœʁ/ 세르뵈르 — 종업원
- Il est serveur dans un café. — 그는 카페 종업원이야.
- La serveuse apporte le café. — 종업원이 커피를 가져와.

**vendeur / vendeuse** /vɑ̃.dœʁ/ 방되르 — 판매원
- Elle est vendeuse. — 그녀는 판매원이야.
- Le vendeur est aimable. — 판매원이 친절해.

**acteur / actrice** /ak.tœʁ/ 악퇴르 — 배우 ≈ actor
- C'est un acteur célèbre. — 유명한 배우야.
- Elle est actrice. — 그녀는 배우야.

**chanteur / chanteuse** /ʃɑ̃.tœʁ/ 샹퇴르 — 가수
- Il est chanteur. — 그는 가수야.
- Ma chanteuse préférée est française. — 내가 좋아하는 가수는 프랑스인이야.

**boulanger / boulangère** /bu.lɑ̃.ʒe/ 불랑제 — 제빵사
- Le boulanger fait du bon pain. — 제빵사가 맛있는 빵을 만들어.
- Elle est boulangère. — 그녀는 제빵사야.

**employé / employée** /ɑ̃.plwa.je/ 앙플루아예 — 회사원, 직원 ≈ employee
- Je suis employé de bureau. — 나는 사무직이야.
- Elle est employée dans une banque. — 그녀는 은행 직원이야.

**docteur / docteure** /dɔk.tœʁ/ 독퇴르 — 의사 (구어) ≈ doctor
- Bonjour, docteur. — 안녕하세요, 의사 선생님.
- Le docteur arrive. — 의사가 와.
