# 색깔 (Les couleurs)

색깔은 형용사라 명사의 성·수에 일치([[12-형용사]]). 대개 명사 뒤에 온다.

**blanc / blanche** /blɑ̃/ 블랑 — 흰
- Je porte un pull blanc. — 흰 스웨터를 입어.
- Elle a une robe blanche. — 그녀는 흰 원피스를 입었어.

**noir / noire** /nwaʁ/ 누아르 — 검은
- J'ai un chat noir. — 검은 고양이가 있어.
- C'est une voiture noire. — 검은 차야.

**rouge** /ʁuʒ/ 루주 — 빨간
- Une pomme rouge. — 빨간 사과.
- J'écris avec un stylo rouge. — 빨간 펜으로 써.

**bleu / bleue** /blø/ 블뢰 — 파란
- Le ciel est bleu. — 하늘이 파래.
- Elle porte une jupe bleue. — 그녀는 파란 치마를 입어.

**vert / verte** /vɛʁ/ 베르 — 초록
- Le feu est vert. — 신호등이 초록불이야.
- J'ai une plante verte. — 초록 식물이 있어.

**jaune** /ʒon/ 존 — 노란
- Un citron jaune. — 노란 레몬.
- Le soleil est jaune. — 해는 노랗다.

**orange** /ɔ.ʁɑ̃ʒ/ 오랑주 — 주황 (불변)
- Un fruit orange. — 주황색 과일.
- Des sacs orange. — 주황색 가방들. (복수도 불변)

**rose** /ʁoz/ 로즈 — 분홍
- Une fleur rose. — 분홍 꽃.
- J'aime le pull rose. — 나는 분홍 스웨터가 좋아.

**violet / violette** /vjɔ.lɛ/ 비올레 — 보라
- Un raisin violet. — 보라색 포도.
- Une écharpe violette. — 보라색 목도리.

**gris / grise** /ɡʁi/ 그리 — 회색
- Le ciel est gris aujourd'hui. — 오늘 하늘이 흐려.
- Il porte une veste grise. — 그는 회색 재킷을 입어.

**marron** /ma.ʁɔ̃/ 마롱 — 갈색 (불변)
- Elle a les yeux marron. — 그녀는 갈색 눈이야.
- Un sac marron. — 갈색 가방.

> [!note] 불변 색
> orange, marron은 성·수에 따라 변하지 않는다: des sacs **orange**.
