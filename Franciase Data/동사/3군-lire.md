# lire /liʁ/ — 읽다

> 3군 불규칙 동사.

## 현재형 (le présent)
| 인칭 | 활용 | IPA |
|---|---|---|
| je (j') | lis | /li/ |
| tu | lis | /li/ |
| il/elle | lit | /li/ |
| nous | lisons | /li.zɔ̃/ |
| vous | lisez | /li.ze/ |
| ils/elles | lisent | /liz/ |

## 예문 (초급)
- Je lis un livre. — 나는 책을 읽는다.
- Vous lisez le journal ? — 신문 읽으세요?
