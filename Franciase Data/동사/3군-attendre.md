# attendre /a.tɑ̃dʁ/ — 기다리다 ≈ attend

> 3군 불규칙 동사.

## 현재형 (le présent)
| 인칭 | 활용 | IPA |
|---|---|---|
| je (j') | j'attends | /a.tɑ̃/ |
| tu | attends | /a.tɑ̃/ |
| il/elle | attend | /a.tɑ̃/ |
| nous | attendons | /a.tɑ̃.dɔ̃/ |
| vous | attendez | /a.tɑ̃.de/ |
| ils/elles | attendent | /a.tɑ̃d/ |

## 예문 (초급)
- J'attends le bus. — 나는 버스를 기다린다.
- Nous attendons nos amis. — 우리는 친구들을 기다린다.
