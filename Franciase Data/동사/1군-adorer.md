# adorer /adɔʁe/ — 아주 좋아하다 ≈ adore

> 규칙 1군(-er) 동사.

## 현재형 (le présent)
| 인칭 | 활용 | IPA |
|---|---|---|
| je (j') | j'adore | /adɔʁ/ |
| tu | adores | /adɔʁ/ |
| il/elle | adore | /adɔʁ/ |
| nous | adorons | /adɔʁɔ̃/ |
| vous | adorez | /adɔʁe/ |
| ils/elles | adorent | /adɔʁ/ |

## 예문 (초급)
- J'adore le chocolat. — 나는 초콜릿을 아주 좋아한다.
- Elles adorent voyager. — 그녀들은 여행을 아주 좋아한다.
