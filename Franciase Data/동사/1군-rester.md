# rester /ʁɛste/ — 머무르다, 남다 ≈ rest

> 규칙 1군(-er) 동사.

## 현재형 (le présent)
| 인칭 | 활용 | IPA |
|---|---|---|
| je (j') | reste | /ʁɛst/ |
| tu | restes | /ʁɛst/ |
| il/elle | reste | /ʁɛst/ |
| nous | restons | /ʁɛstɔ̃/ |
| vous | restez | /ʁɛste/ |
| ils/elles | restent | /ʁɛst/ |

## 예문 (초급)
- Je reste à la maison ce soir. — 나는 오늘 저녁 집에 머무른다.
- Nous restons deux jours ici. — 우리는 여기 이틀 머무른다.
