# tourner /tuʁ.ne/ — 돌다, 돌리다, 방향을 틀다

> 규칙 1군(-er) 동사.

## 현재형 (le présent)
| 인칭 | 활용 | IPA |
|---|---|---|
| je (j') | tourne | /tuʁn/ |
| tu | tournes | /tuʁn/ |
| il/elle | tourne | /tuʁn/ |
| nous | tournons | /tuʁ.nɔ̃/ |
| vous | tournez | /tuʁ.ne/ |
| ils/elles | tournent | /tuʁn/ |

## 예문 (초급)
- Tourne à droite. — 오른쪽으로 돌아.
- Nous tournons à gauche. — 우리는 왼쪽으로 돈다.
