# vouloir /vu.lwaʁ/ — 원하다, ~하고 싶다

> 3군 불규칙 동사. **vouloir + 명사/동사원형**. 정중한 요청은 조건법 je voudrais(~하고 싶습니다).

## 현재형 (le présent)
| 인칭 | 활용 | IPA |
|---|---|---|
| je (j') | veux | /vø/ |
| tu | veux | /vø/ |
| il/elle | veut | /vø/ |
| nous | voulons | /vu.lɔ̃/ |
| vous | voulez | /vu.le/ |
| ils/elles | veulent | /vœl/ |

## 예문 (초급)
- Je veux un café. — 커피 한 잔 원해요.
- Tu veux manger ? — 먹고 싶어?
