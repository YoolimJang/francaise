# parler /paʁle/ — 말하다

> 규칙 1군(-er) 동사.

## 현재형 (le présent)
| 인칭 | 활용 | IPA |
|---|---|---|
| je (j') | parle | /paʁl/ |
| tu | parles | /paʁl/ |
| il/elle | parle | /paʁl/ |
| nous | parlons | /paʁlɔ̃/ |
| vous | parlez | /paʁle/ |
| ils/elles | parlent | /paʁl/ |

## 예문 (초급)
- Je parle français. — 나는 프랑스어를 말한다.
- Vous parlez trop vite. — 당신은 너무 빨리 말해요.
