# prendre [pʁɑ̃dʁ] — 잡다, (교통·음식을) 취하다

> 불규칙 3군 동사, 교통수단·음식을 이용/주문할 때 씀

## 현재형 (le présent)
| 인칭 | 활용 | IPA |
|---|---|---|
| je (j') | prends | [pʁɑ̃] |
| tu | prends | [pʁɑ̃] |
| il/elle | prend | [pʁɑ̃] |
| nous | prenons | [pʁənɔ̃] |
| vous | prenez | [pʁəne] |
| ils/elles | prennent | [pʁɛn] |

## 예문 (초급)
- Je prends le bus. — 나는 버스를 탄다.
- Je prends un café. — 나는 커피를 마신다(주문한다).
