# dormir /dɔʁ.miʁ/ — 자다

> 3군 불규칙 동사 (partir·sortir 계열).

## 현재형 (le présent)
| 인칭 | 활용 | IPA |
|---|---|---|
| je (j') | dors | /dɔʁ/ |
| tu | dors | /dɔʁ/ |
| il/elle | dort | /dɔʁ/ |
| nous | dormons | /dɔʁ.mɔ̃/ |
| vous | dormez | /dɔʁ.me/ |
| ils/elles | dorment | /dɔʁm/ |

## 예문 (초급)
- Je dors bien. — 나는 잘 잔다.
- Les enfants dorment. — 아이들이 잔다.
