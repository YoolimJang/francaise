# 동사 + 동사원형 (Verbe + infinitif)

일부 동사는 뒤에 **동사원형(infinitif)**을 바로, 또는 전치사(à/de)와 함께 취한다.

## 전치사 없이 + 동사원형
| 동사 | 뜻 | 예문 | 예문 뜻 |
|---|---|---|---|
| aller + inf. | ~할 것이다 (근접미래) | Je **vais manger**. | 먹을 거야 |
| devoir + inf. | ~해야 한다 | Je **dois partir**. | 떠나야 해 |
| aimer + inf. | ~하기를 좋아하다 | J'**aime voyager**. | 여행을 좋아해 |
| adorer + inf. | ~하기를 아주 좋아하다 | J'**adore chanter**. | 노래를 정말 좋아해 |
| préférer + inf. | ~하기를 선호하다 | Je **préfère rester**. | 남는 게 좋아 |
| détester + inf. | ~하기를 싫어하다 | Je **déteste attendre**. | 기다리기 싫어 |
| vouloir + inf. | ~하고 싶다 | Je **veux dormir**. | 자고 싶어 |
| pouvoir + inf. | ~할 수 있다 | Je **peux venir**. | 갈 수 있어 |

## 전치사 de + 동사원형
| 동사 | 뜻 | 예문 | 예문 뜻 |
|---|---|---|---|
| essayer **de** + inf. | ~하려고 애쓰다 | J'**essaie de** comprendre. | 이해하려 애써 |
| partir는 자체로 이동 | | | |

> [!note] 근접미래 (futur proche)
> **aller(현재형) + 동사원형** = 가까운 미래. Je vais / tu vas / il va + inf.
> - Je **vais** téléphoner. (전화할 거야.)
> - Nous **allons** manger. (우리 먹을 거야.)
