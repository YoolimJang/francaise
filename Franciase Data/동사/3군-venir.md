# venir /və.niʁ/ — 오다

> 3군 불규칙 동사. **venir de + 장소** = ~에서 오다 / **venir à** = ~에 오다.

## 현재형 (le présent)
| 인칭 | 활용 | IPA |
|---|---|---|
| je (j') | viens | /vjɛ̃/ |
| tu | viens | /vjɛ̃/ |
| il/elle | vient | /vjɛ̃/ |
| nous | venons | /və.nɔ̃/ |
| vous | venez | /və.ne/ |
| ils/elles | viennent | /vjɛn/ |

## 예문 (초급)
- Je viens de Corée. — 나는 한국에서 왔다.
- Tu viens avec moi ? — 나랑 같이 올래?
