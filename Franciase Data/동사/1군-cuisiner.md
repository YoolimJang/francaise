# cuisiner /kɥizine/ — 요리하다 ≈ cuisine

> 규칙 1군(-er) 동사.

## 현재형 (le présent)
| 인칭 | 활용 | IPA |
|---|---|---|
| je (j') | cuisine | /kɥizin/ |
| tu | cuisines | /kɥizin/ |
| il/elle | cuisine | /kɥizin/ |
| nous | cuisinons | /kɥizinɔ̃/ |
| vous | cuisinez | /kɥizine/ |
| ils/elles | cuisinent | /kɥizin/ |

## 예문 (초급)
- Je cuisine ce soir. — 나는 오늘 저녁에 요리한다.
- Elle cuisine très bien. — 그녀는 요리를 아주 잘한다.
