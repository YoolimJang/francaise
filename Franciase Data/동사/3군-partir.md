# partir /paʁ.tiʁ/ — 떠나다, 출발하다

> 3군 불규칙 동사 (dormir·sortir 계열).

## 현재형 (le présent)
| 인칭 | 활용 | IPA |
|---|---|---|
| je (j') | pars | /paʁ/ |
| tu | pars | /paʁ/ |
| il/elle | part | /paʁ/ |
| nous | partons | /paʁ.tɔ̃/ |
| vous | partez | /paʁ.te/ |
| ils/elles | partent | /paʁt/ |

## 예문 (초급)
- Je pars demain. — 나는 내일 떠난다.
- Nous partons en vacances. — 우리는 휴가를 떠난다.
