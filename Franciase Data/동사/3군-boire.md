# boire /bwaʁ/ — 마시다

> 3군 불규칙 동사. 목적어엔 흔히 부분관사(du/de la/de l') 사용.

## 현재형 (le présent)
| 인칭 | 활용 | IPA |
|---|---|---|
| je (j') | bois | /bwa/ |
| tu | bois | /bwa/ |
| il/elle | boit | /bwa/ |
| nous | buvons | /by.vɔ̃/ |
| vous | buvez | /by.ve/ |
| ils/elles | boivent | /bwav/ |

## 예문 (초급)
- Je bois de l'eau. — 나는 물을 마신다.
- Il boit du café. — 그는 커피를 마신다.
