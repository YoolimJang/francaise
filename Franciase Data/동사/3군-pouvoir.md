# pouvoir /pu.vwaʁ/ — 할 수 있다

> 3군 불규칙 동사. **pouvoir + 동사원형**. 정중한 요청: je peux ... ?

## 현재형 (le présent)
| 인칭 | 활용 | IPA |
|---|---|---|
| je (j') | peux | /pø/ |
| tu | peux | /pø/ |
| il/elle | peut | /pø/ |
| nous | pouvons | /pu.vɔ̃/ |
| vous | pouvez | /pu.ve/ |
| ils/elles | peuvent | /pœv/ |

## 예문 (초급)
- Je peux venir. — 나는 갈 수 있어.
- Tu peux m'aider ? — 나 좀 도와줄 수 있어?
