# 핵심 표현 (Expressions clés) — MOC

책 진도에 맞춘 과별 회화 표현. 각 표현엔 예문 2개. 문법 규칙은 문법 노트로 링크.

| 과 | 주제 | 핵심 |
|---|---|---|
| [[01과-기본인사]] | 기본 인사·예의 | bonjour, merci, ça va ? |
| [[02과-이름말하기]] | 이름·자기소개 | Je m'appelle, Je suis étudiant |
| [[03과-의문문]] | 의문문·부정문 | Est-ce que, ne... pas |
| [[04과-나이와기호]] | 나이·기호 | J'ai 20 ans, J'aime la musique |
| [[05과-사물묘사]] | 사물 묘사·존재 | Qu'est-ce que c'est ?, il y a |
| [[06과-가족과사람묘사]] | 가족·사람·감정 | Qui est-ce ?, Il est comment ? |
| [[07과-시간과행선지]] | 시간·행선지 | Il est quelle heure ?, Je vais au... |
| [[08과-가까운미래와날짜]] | 근접미래·날짜 | Je vais partir, On est quel jour ? |
| [[09과-날씨와감각]] | 날씨·감각 | Il fait beau, J'ai froid |
| [[10과-취미와의무]] | 취미·의무 | J'aime faire du sport, Je dois... |
| [[11과-주문과계산]] | 주문·계산 | Je voudrais..., Ça coûte combien ? |
| [[12과-허락과비교와명령]] | 허락·비교·명령 | Je peux...?, plus... que, 명령법 |
| [[13과-길묻기와안내]] | 길 묻기·안내 | Où est...?, Allez tout droit |
