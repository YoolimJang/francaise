# 4과. 나이·기호 말하기 (L'âge et les goûts)

## 나이 묻고 답하기
**Vous avez quel âge ? / Tu as quel âge ?** /kɛ.l‿aʒ/ — 몇 살이세요? / 몇 살이야?
- Vous avez quel âge ? — J'ai trente ans. — 몇 살이세요? — 서른이에요.
- Tu as quel âge ? — 너 몇 살이야?

**J'ai + 숫자 + an(s)** /ʒe ... ɑ̃/ — 저는 ~살이에요 (avoir + 숫자 + an)
- J'ai vingt ans. — 저는 스무 살이에요.
- Il a dix ans. — 그는 열 살이에요.

## 기호(좋아하는 것) 말하기
**J'aime + 명사** /ʒɛm/ — ~을 좋아하다 (aimer + 명사)
- J'aime la musique coréenne. — 저는 한국 음악을 좋아해요.
- J'aime le café. — 저는 커피를 좋아해요.

**Je n'aime pas + 명사** — ~을 안 좋아하다
- Je n'aime pas le sport. — 저는 운동을 안 좋아해요.
- Je n'aime pas les maths. — 저는 수학을 안 좋아해요.
